# cpp project skeleton
c++ project skeleton, use c++14 standard for generic use. it's a good template for starter.
this code base use googletest for unit testing, and benchmark for benchmark tesing.



## Build Archieture
- x86_64

## Build Compiler

## Requirements
- googletest
- benchmark


## Running App
```
cd bin && ./app

```


## Running Test
```bash
cd bin && ./apptest

```

## Running Benchmark
```bash
cd bin && ./appbenchmark
```
